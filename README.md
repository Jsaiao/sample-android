# OmiseGO Android sample

The OmiseGO Android sample app is a simple app to demonstrate the use of the [Android SDK](https://github.com/omisego/android-sdk)

# Requirements

- Android 5.0+
- Android Studio 3.0+
- Kotlin 1.2+

# Tests

Run the tests from the Android Studio
